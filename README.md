# LNH Enterprise

**Sell Smarter & Grow Faster** — LNH Enterprise partners with brands to maximize their Amazon potential: listing optimization, advertising, distribution, and brand protection. Boston, MA.

## Tech Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- Google Fonts: Cormorant Garamond, DM Sans

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint

## Project Structure

- `src/app/` — Pages and layout (home, grow, distribute, protect, optimization, mission, contact)
- `src/components/` — Nav, Footer, Logo, Marquee, Reveal, ContactForm

## Deploy

Deploy to [Vercel](https://vercel.com) by connecting this repo. The project is configured for static/SSG where possible.
